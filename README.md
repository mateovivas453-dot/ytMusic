# 📺 StreamNode: Next-Gen Video Interface

**StreamNode** es una plataforma de visualización de videos con una interfaz moderna y atractiva. Su objetivo es ofrecer una experiencia visual diferente, rápida y agradable para el usuario.

---

## 🚀 Clonar el proyecto

Para descargar el proyecto en tu computadora, usa este comando:

```bash
git clone https://github.com/tu-usuario/streamnode.git
````

---

## 💡 Descripción

El proyecto permite navegar entre videos de forma sencilla, con una interfaz limpia y efectos visuales suaves.

---

## 🛠️ Tecnologías usadas

* HTML5
* CSS3
* Grid y Flexbox
* Iconos SVG
